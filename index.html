Process to connect git hub

  1. Create a new repository in github
  2. Create a folder similiarly the repository in your computer
  3. Open the folder in vs code
  4. press ctrl+j for terminal
  5. init git
  6. git add .
  7. git commit -m"first commit"
  8. git branch -M main
  9. git remote add origin https://github.com/indrojitmondal/my-first-github-repo.git(It will show in git hub)
  10. git push -u origin main

Process to 2nd or more commit

  1. git add .
  2. git commit -m"meaningfull message"
  3. git push
